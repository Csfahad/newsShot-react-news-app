## Installation

Clone this repo

```bash
  git clone https://github.com/<YOUR_USERNAME>/newsShot-react-news-app.git
  cd newsShot-react-news-app
```

Install the node_modules

```bash
  npm install
```

run the project

```bash
  npm run start
```
